# ⚙️ XP-System & Regeln

---

## 📐 Die 1%-Methode im Spiel

> **1% besser pro Tag = nach 365 Tagen 37× besser.**

Jede abgeschlossene Sidequest ist ein kleiner Zinseszins. Das Spiel visualisiert, was unsichtbar passiert: dein kumuliertes Wachstum.

| Tage konsequent | Multiplikator |
|-----------------|---------------|
| 1–6 Tage | ×1,0 (normal) |
| 7–13 Tage | ×1,1 (Streak-Bonus) |
| 14–29 Tage | ×1,2 |
| 30+ Tage | ×1,5 |

> Streak = nie zweimal hintereinander aussetzen. Einmal auslassen = Versehen. Zweimal = Streak bricht.

---

## 📊 XP-Zuweisung pro Bereich

Jede XP-Einheit zählt **doppelt**: einmal zum Gesamt-Level, einmal zum Sub-Level des Bereichs.

- Training absolviert → +30 XP Gesamt, +30 XP Körper
- ETF-Sparplan läuft → +10 XP Gesamt, +10 XP Finanzen

---

## ❌ Strafen (optional, milde)

| Ereignis | Effekt |
|----------|--------|
| Streak bricht | Streak-Bonus zurück auf ×1,0 |
| Zweimal hintereinander ausgesetzt | –10 XP im entsprechenden Bereich |
| Main-Quest-Deadline verpasst | Keine XP für diesen Meilenstein |

> [!tip] Regel von James Clear
> Niemals zweimal hintereinander aussetzen. Ein schlechter Tag ist okay. Zwei schlechte Tage in Folge sind eine neue (schlechte) Gewohnheit.
