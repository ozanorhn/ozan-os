# CODE-Methode – Capture, Organize, Distill, Express

> [!quote] Kern
> Schwäche in einer Phase blockiert die nächste. Die meisten scheitern an Express.

## Die 4 Phasen

| Phase | Frage |
|-------|-------|
| **Capture** | Was resoniert? |
| **Organize** | Wo gehört es hin (PARA)? |
| **Distill** | Was ist die Essenz? |
| **Express** | Was tue ich damit in der Welt? |

CODE ist das **Was** der zweiten Gehirne. PARA (siehe [[PARA-Struktur - Organisation nach Aktionierbarkeit]]) ist das **Wo**.

## Diagnose

- **Capture-Schwäche:** zu viel oder zu wenig erfasst – kein Resonanz-Filter
- **Organize-Schwäche:** Inbox quillt über, nichts hat seinen Platz
- **Distill-Schwäche:** Notizen sind reine Kopien, keine eigene Sprache
- **Express-Schwäche:** das System läuft, aber nichts kommt in der Welt an

> Wer captured, organized und distilled aber nicht expressed, baut ein Museum.

## Anwendung

- Wöchentliches Review (Sonntag): bei welcher Phase staut es gerade?
- Bei jeder neuen Notiz die Express-Frage: *Was tue ich heute damit?*
- Wenn keine Express-Antwort kommt → vermutlich Collector's Fallacy.
