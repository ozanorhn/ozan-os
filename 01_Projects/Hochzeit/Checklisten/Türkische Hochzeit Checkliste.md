# Türkische Hochzeit – Checkliste

> [!info] Überblick
> Alle Schritte für eine türkische Hochzeit, von der Verlobung bis zur Düğün.

## Phasen der türkischen Hochzeit

```
Söz Kesme → Nişan (Verlobungsfeier) → Kına Gecesi → Nikah → Düğün
```

---
